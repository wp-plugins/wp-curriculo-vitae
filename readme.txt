== Description ==

**Portugues**

O **WP-Curriculo Vitae Free** e um plugin que permite que usuarios a cadastrem seu curriculo no site para divulgacao online ou para uso do site.

O plugin trabalha com duas versoes:
 
**Versao gratuita:**
 
1. Cadastrar o curriculo;
2. Listagem de curriculos com sistema de busca.
 
**Versao paga:**
 
1. Cadastrar o curriculo;
2. Editar atraves de area restrita;
3. Widget de login na area restritra;
4. Listagem de curriculos com sistema de busca;
5. Widget de Busca de Curriculos.

Para comprar a versao paga acesse a pagina [Wiliamluis](http://wiliamluis.wordpress.com/plugin/) ou acesse pelo menu da versao gratuita.
 

**Como utilizar**
 
Para exibir o formulario de cadastro, crie uma pagina e coloque o shortcode **[formCadastro]**.
 
Para exibir a listagem dos cadastrados, crie uma pagina e coloque o shortcode **[listCurriculos]**.
 
Use o widget **WP-Curriculo Vitae busca** para exibir a busca rapida de curriculo em todas as paginas.
 
Use o widget **WP-Curriculo Vitae login** para exibir o formulario de login em todas as paginas.
 
*Obs.* Os widget sao apenas para a versao paga.

Desenvolvido por [Wiliamluis](http://www.wiliamluis.wordpress.com) - [Atualnet](http://www.atualnet.com.br)

**English**

The WP-Free Curriculum Vitae  and a plugin that allows users to register with the site to your resume online disclosure or use of the site.

The plugin works with two versions:

**Free Version:**

1. Register the curriculum;
2. Listing curricula with search system;

**Paid version:**

1. Register the curriculum;
2. Edit through the restricted area;
3. Widget login restritra in area;
4. Listing curricula with search system;
5. Search Resumes widget.

To purchase the paid version go to page [Wiliamluis] (http://wiliamluis.wordpress.com/plugin/) or access via the menu version free.

**How to use**

To view the registration form, create a page and put the shortcode ** [formCadastro] **.

To view the list of registered, create a page and put the shortcode ** [listCurriculos] **.
 
Use the widget ** WP-Curriculum Vitae  busca** to display the search quickly to resume on all pages.
 
Use the widget ** WP-Curriculum Vitae  login ** to display the login form on every page.
 
* Note * The widget are only for the paid version.

Developed by [Wiliamluis] (http://www.wiliamluis.wordpress.com) - [Atualnet] (http://www.atualnet.com.br)
